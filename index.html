지역별 회원 게시판 2023-05-02 ~ 2023-05-28

프로젝트 개요 개발내용
제목 : 지역별 회원 게시판 - 배포중
기간 : 2023-05-02 ~ 2023-05-28
인원 : 1명
 - 모델1 방식을 이용하여 지역별 회원 게시판 구현
- 로그인, 로그아웃 기능 구현
- 회원가입 및 회원정보 수정, 회원탈퇴 구현
- 10페이지 단위의 페이징 구현
담당기능
  1. 회원
    • Session
      - 세션 기반의 로그인/로그아웃 기능을 구현하였습니다.
    • 회원가입
      - 로그아웃 상태에서만 회원가입 메뉴가 표시됩니다.
      - 중복된 아이디로는 가입할 수 없습니다.
    • 프로필
      - 로그인 상태에서만 프로필 메뉴가 표시됩니다.
      - 회원정보 수정/탈퇴 시 비밀번호가 일치해야 합니다.
  2. 카테고리
    • 카테고리 메뉴
      - 카테고리별 게시글의 수를 카운트하여 확인할 수 있도록 GROUP BY절을 사용했습니다.
      - 전체 게시글의 수도 함께 출력하기 위해 UNION ALL을 이용하여 조회했습니다.
      - 카테고리 클릭 시 해당 카테고리의 게시글만 조회할 수 있습니다.
    • 카테고리 관리
      - 로그인 상태에서만 카테고리 관리 버튼이 표시됩니다.
      - 게시글이 존재하는 카테고리는 수정/삭제할 수 없도록 오류메세지가 표시됩니다.
      - 중복되는 이름의 카테고리는 생성할 수 없습니다.
  3. 게시판
    • 게시글/댓글 관리
      - 로그인 상태에서만 글쓰기 버튼과 댓글 입력란이 표시됩니다.
      - 로그인 아이디와 작성자 아이디가 일치하는 경우에만 게시글/댓글의 수정/삭제 버튼이 표시됩니다.
    • 페이지 번호 페이징
      - 10페이지 단위의 페이지바를 만들어 더욱 직관적인 페이징을 구현했습니다.
      - 카테고리 클릭 후 페이징시에도 카테고리 선택이 유지됩니다.
      - 댓글 목록도 페이징이 적용됩니다.

- 게시글은 간단한 게시판 형태로 확인 가능
담당기능
  1. 달력
    • Calendar API 사용
      - JAVA의 기본 라이브러리 중 하나인 Calendar API를 사용하여 달력을 구현했습니다.
    • 일정 미리보기
      - 달력의 해당 날짜의 일정을 작성시 선택한 색상과 함께 미리 보여줍니다. (5글자 제한)
      - 날짜 클릭 시 해당 날짜의 일정 목록과 일정 추가가 가능한 페이지로 이동합니다.

    • 시작 페이지
      - 홈에서 최근 게시글 목록을 확인할 수 있습니다.
      - 홈에서 오늘 날짜의 일정 목록을 확인할 수 있습니다.
    • 일정/게시글 관리
      - 일정 메모는 색상 선택이 가능하며, 조회 시 해당 색상으로 출력됩니다.
      - 게시판은 이전/다음 버튼을 이용한 간단한 페이징이 가능합니다.
      - DML을 이용하여 일정/게시글 CRUD가 가능하도록 했습니다.
      - 일정/게시글을 수정 또는 삭제하려면 작성시 입력한 비밀번호가 일치해야 합니다.
  3. CSS
    • Bootstrap5
      - Bootstrap5를 이용하여 간단한 CSS 작업을 했습니다.
